Hoisting

The variable goes before the method, calling the add function.

console.log(add(2, 8));

function add(x, y) {

return x +y;

}



Anonymous

A function without a name, where the argument can be used in conjunction with other functions, or as an immediately execution.

const addInAVariable = function (x, y) {

return x + y;

}

console.log(addInAVariable(2, 5));



Lambda

A function where the {} operator is being replace by the arrow => which creates shorter syntax.

const addArrowFunction = (x, y) => (35 ,2);

console.log(addArrowFunction());



Callback

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

function doActionWithSomeone(anyFunctionReference, name) {

anyFunctionReference(name);

}

const running = (name) => console.log(`${name} is running`);

doActionWithSomeone(running, "Kev");
